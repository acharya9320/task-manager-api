# 🚀 Task Management API (Django + DRF + JWT)

A production-like backend project built using Django and Django REST Framework.

---

## 📌 Features

- 🔐 JWT Authentication (Login system)
- ✅ CRUD Operations (Create, Read, Update, Delete tasks)
- 👥 Role-based Access (Admin vs User)
- 🔍 Filtering (status, assigned user)
- 📄 Pagination (Page-wise data)
- 🧪 Unit Testing (5+ tests)

---

## 🛠️ Tech Stack

- Python
- Django
- Django REST Framework
- SimpleJWT
- SQLite
